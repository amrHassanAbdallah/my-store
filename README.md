# MyStore

This project is a single page app on top of angular.

### Features
* List products
* View product details
- Add product to cart
- Checkout products
- Remove product from cart
### Dependencies
* [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
## Up & Running
1.  ```
    npm i
    ```
1.  ```
    ng serve
    ```
